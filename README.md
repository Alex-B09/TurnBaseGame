# Introduction 
This is a test project to see how to use the GAS (Gameplay Ability System) of Unreal Engine.
I have made both C++ and Blueprint abilities and effects just to get a feel of how everything works together

# Controls
Keyboard:
 - A action
 - S cancel
 - Arrows controls
